# Lesson Package Generator 사용자 매뉴얼

> 교사를 위한 단계별 사용 가이드. 코딩 지식이 없어도 따라 할 수 있도록 작성되었습니다.
> 기본 사용 경로는 **웹앱**입니다. 개발/자동화용 CLI는 §8에 별도로 있습니다.

---

## 1. 가장 쉬운 사용법 (5분 요약)

```
1) 웹앱을 연다            → lesson-package-generator/webapp/index.html
2) API 키를 넣는다 (1회)  → 우측 상단 ⚙️ → 키 붙여넣기 → 저장
3) 입력한다              → 본문 · 테마 · 대상 · 분량
4) ✨ 생성하기           → 진행 막대가 5단계를 차례로 표시
5) 결과를 쓴다           → 탭에서 보고, 다운로드/프롬프트 복사
```

처음 한 번만 API 키를 넣으면, 이후에는 입력 → 생성만 반복하면 됩니다.

---

## 2. API 키 설정 (최초 1회)

웹앱은 회원님의 브라우저에서 **직접** Claude(Anthropic)에 요청합니다. 그래서 API 키가 필요합니다.

### 2.1 키 발급
1. https://console.anthropic.com 접속 → 로그인
2. **Settings → API Keys → Create Key**
3. 생성된 `sk-ant-...` 문자열을 복사 (이 화면을 벗어나면 다시 볼 수 없으니 주의)

### 2.2 키 입력
1. 웹앱 우측 상단 **⚙️** 클릭
2. **Anthropic API 키** 칸에 붙여넣기
3. (선택) **모델** 선택 — Sonnet 4.6(권장·빠름) / Opus 4.8(최고 품질·느림) / Haiku 4.5(경량)
4. **저장**

### 2.3 키는 어디에 저장되나요? (보안)
- 키는 **이 브라우저(localStorage)에만** 저장됩니다. 어떤 서버로도 전송되지 않습니다.
- 코드에 키가 박혀 있지 않으므로, 웹앱을 배포해도 키가 노출되지 않습니다.
- **주의**: 공용 PC에서는 사용 후 ⚙️에서 키를 지우거나 브라우저 데이터를 비우세요. 다수가 쓰는 공개 서비스로 만들 때는 별도 서버(프록시)가 필요합니다.

## 3. 입력 작성 가이드

| 칸 | 필수 | 작성 요령 | 예시 |
|----|------|----------|------|
| **본문** | 예 | 성경 본문 구절(장:절)이나 본문 요약. 길어도 됩니다. | `누가복음 8:26-39 — 거라사 군대귀신 들린 사람을 고치신 사건` |
| **테마 / 강조점** | 예 | 이번 수업에서 가장 강조할 한 단어/구절 | `자유` / `은혜` / `회복` |
| **대상** | 예 | 연령·인원·맥락 | `중등부 20명, 주일 오후` |
| **분량(분)** | 예 | 총 진행 시간(분) | `80` |
| **부가물 선택** | — | 만들 부가물 체크 (수업안은 항상 생성) | 교보재 ☑ 찬양 ☑ 홍보영상 ☑ |

> 팁: 본문이 구체적일수록(구절 + 짧은 요약) 결과 품질이 올라갑니다. 테마는 추상어 한 단어보다 "정죄 없는 자유"처럼 방향이 담기면 좋습니다.

---

## 4. 생성 진행 보기

**✨ 생성하기**를 누르면 5단계가 순서대로 진행됩니다:

```
① 수업안 설계   ⟳ 생성 중 → ✓ 8개 섹션 완성
② 교보재        ⟳ → ✓ 도입게임·토의·활동·워크시트
③ 찬양          ⟳ → ✓ 오리지널 가사 + Suno 프롬프트
④ 홍보영상      ⟳ → ✓ 스토리보드 + 컷별 프롬프트
⑤ 자기검수      ⟳ → ✓ 판정 PASS · 통과 13/13
```

- 부가물을 체크 해제하면 해당 단계는 "건너뜀"으로 표시됩니다.
- 중간에 오류가 나면 해당 단계에 ✗ 와 메시지가 표시됩니다. **처음으로**를 눌러 다시 시도하세요(키/네트워크 확인).

---

## 5. 결과 활용 — 탭별 안내

생성이 끝나면 결과가 탭으로 나타납니다. 상단에 대상·분량·자기검수 판정이 요약됩니다.

### 5.1 수업안 탭 (가장 중요)
학습목표·도입·본문전개·핵심메시지·적용·토의질문·마무리·시간배분 8개 섹션. **이 탭만으로 수업을 진행할 수 있습니다.**
- **⬇ JSON** / **⬇ MD**: 수업안 저장(MD는 문서로 바로 읽기 좋음).

### 5.2 교보재 탭
도입게임·토의·활동·워크시트·슬라이드. 이미지가 필요한 곳은 `[IMG: …]` 프롬프트로 모아 보여줍니다(§6.3).
- **⬇ JSON** / 이미지 프롬프트 **📋 복사**.

### 5.3 찬양 탭
오리지널 가사(1절·후렴·2절·브릿지) + **Suno 음악 프롬프트**.
- **⬇ JSON** / **⬇ 가사 MD** / 프롬프트 **📋 복사** → §6.1로 음원 만들기.

### 5.4 홍보영상 탭
전체 내레이션 + 스토리보드(컷별 장면·자막·영상/이미지 프롬프트).
- **⬇ JSON** / **⬇ 자막 SRT** / 컷별 프롬프트 **📋 복사** → §6.2로 영상 만들기.

### 5.5 자기검수 탭
PK1–PK13 검사 결과와 판정(PASS/FAIL). 경고가 있어도 치명 항목이 통과면 PASS입니다. 어떤 항목이 왜 걸렸는지 비고로 보여줍니다.

### 5.6 전체 저장
상단 **⬇ 전체(JSON)**: 수업안 + 부가물 + 자기검수를 하나의 JSON으로 저장.

## 6. 외부 AI 핸드오프 (음원·영상·이미지)

이 앱은 음원과 영상을 **직접 만들지 않습니다**. 대신 외부 AI에 넣을 **정확한 프롬프트**를 줍니다. 아래 순서로 완성하세요.

### 6.1 찬양 → 음원 (Suno)
1. 찬양 탭에서 **Suno 음악 프롬프트 📋 복사**
2. https://suno.com 접속 → **Custom 모드**
3. 가사 칸: 찬양 탭의 가사(⬇ 가사 MD)를 붙여넣기
4. 스타일/프롬프트 칸: 복사한 프롬프트 붙여넣기 → 생성
5. 마음에 드는 음원을 내려받아 수업/예배에 사용
> 가사는 오리지널 창작물입니다. 외부 배포(공연·녹음 공개) 전 CCLI 등 권리 확인이 필요합니다.

### 6.2 홍보영상 → 영상 (Runway / Pika / Kling 등)
1. 홍보영상 탭의 각 **Cut**에서 🎬 **영상 프롬프트 복사**
2. 영상 생성 AI(Runway/Pika/Kling 등)에 컷별로 붙여넣어 클립 생성
   - 영상 AI가 어려우면 🖼 **이미지 프롬프트**로 정지컷을 만들어 슬라이드쇼로 대체
3. 컷들을 순서대로 편집(또는 CLI의 `assemble_promo_video.py`로 ffmpeg 조립)
4. **⬇ 자막 SRT**를 편집 도구에 올려 자막 입히기
5. 내레이션은 전체 내레이션 텍스트를 TTS 또는 직접 녹음

### 6.3 교보재 이미지 → 이미지 AI
1. 교보재 탭의 **이미지 프롬프트 📋 복사**
2. 이미지 생성 AI(예: 사내 도구/외부 서비스)에 붙여넣어 삽화 생성
3. 워크시트·슬라이드에 배치

## 8. (선택) CLI 사용법 — 개발/자동화

웹앱과 동일한 파이프라인을 명령줄에서 실행합니다.

```bash
cd lesson-package-generator

# 수업안만
python scripts/orchestrator.py --body "누가복음 8:26-39" --audience "중등부" --volume 80 --emphasis "자유"

# 부가물 포함 + 비대화형(휴먼 게이트 자동 승인)
python scripts/orchestrator.py --body "..." --audience "..." --volume 80 --emphasis "자유" \
  --with-teaching --with-praise --with-promo --auto-approve

# 검증
python scripts/validate_lesson_plan.py
python scripts/validate_package_integrity.py
```

- API 키 없이 실행하면 **placeholder 모드**(구조 데모용 결정론적 산출). 실제 콘텐츠는 `ANTHROPIC_API_KEY` 설정 + `LESSON_PACKAGE_PLACEHOLDER=0`에서 생성됩니다.
- 산출물은 `outputs/`(또는 `--project-dir`)에 저장됩니다.

## 9. 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| "API 키가 설정되지 않았습니다" | 키 미입력 | ⚙️에서 키 입력 |
| "API 키가 올바르지 않습니다 (401)" | 키 오타/만료 | console.anthropic.com에서 키 재확인·재발급 |
| "요청 한도를 초과했습니다 (429)" | 사용량 초과 | 잠시 후 재시도 / 결제·한도 확인 |
| "네트워크 오류 또는 CORS 차단" | 오프라인 또는 브라우저 정책 | 인터넷 확인. `file://`로 막히면 `webapp/`에서 `python -m http.server` 후 `localhost:8000`으로 열기 |
| 결과가 본문과 동떨어짐 | 입력이 모호함 | 본문에 구절+요약, 테마에 방향을 더 구체적으로 |
| 진행 중 ✗ 표시 | 일시 오류 | **처음으로** → 다시 생성 |

## 10. 용어 사전

| 용어 | 뜻 |
|------|----|
| 수업안(lesson plan) | 핵심 산출물. 8개 섹션으로 된 가르칠 수 있는 수업 계획 |
| 부가물(supplementary) | 교보재·찬양·홍보영상. 수업안에서 파생되는 선택 산출물 |
| 자기검수(self-check) | PK1–PK13 결정론적 무결성 검사. PASS/FAIL 판정 |
| Suno | 가사+프롬프트로 음원을 만드는 외부 AI |
| SRT | 영상 자막 파일 형식 |
| API 키 | Claude 호출에 필요한 개인 인증 문자열 (`sk-ant-...`) |
| localStorage | 브라우저에 데이터를 저장하는 공간(키 저장 위치) |
