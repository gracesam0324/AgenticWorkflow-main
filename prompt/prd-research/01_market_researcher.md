# Market Researcher 분석
> AI Agentic Workflow Automation System (로컬 실행)
> 조사일: 2026-04-07

---

## Executive Summary

**현재 시장 상황**: AI coding automation 시장은 급성장 중 (CAGR 24-27%, 2030년 $50B 추정)이나, 이미 **Claude Code와 Cursor가 강력한 포지션**을 확보한 상태. 로컬 실행 모델은 **틈새 시장(niche)으로 기회가 제한적**이나, **보안/프라이버시/오프라인 요구 기업들**에게는 차별화된 가치가 있음.

---

## BRANCH A: 기회주의적 관점 (낙관론)

### 1. 시장 규모

| 지표 | 수치 |
|------|------|
| 2024년 AI Coding 시장 | $5.5B ~ $12.26B |
| 2030년 예상 규모 | $50B |
| CAGR | 24-27% |
| AI Agent 단독 시장 | $7.6B (2025) → $50B+ (2030) |

### 2. 경쟁사 약점

**Claude Code**: 높은 비용($100-200/개발자/월), 데이터 프라이버시 리스크, 레이트 제한, 기업 보안 우려  
**Cursor**: IDE 종속성(VS Code), 클라우드 의존, 오프라인 불가  
**Devin AI**: 70% 실패율, 12-15분 응답 지연, 모호한 태스크에서 성능 급락

### 3. 로컬 실행 차별화 강점

| 강점 | 타겟 고객 |
|------|---------|
| 프라이버시 (코드 외부 전송 Zero) | 금융/방위/정부 기관 |
| 비용 구조 (일회성 설치) | 스타트업, 중소기업 |
| 오프라인 작업 가능 | IoT/임베디드 팀 |
| EU AI Act 규정 준수 | 유럽 기업 |
| 레이트 제한 없음 | DevOps, CI/CD |

### 4. 지금이 진입 황금기인 이유

1. EU AI Act 2026년 8월 본격 시행
2. "Claude Code leak" 이후 기업 신뢰도 하락
3. Cline 1.5M 사용자 → 로컬 수요 검증
4. 멀티-에이전트 시스템이 2026년 주요 트렌드
5. Claude Code 비용 압박 (100인 팀 → $2.4M/년)

### 5. 반드시 포함되어야 할 기능 Top 3 (Branch A)

1. 로컬-전용 모드 + 온디바이스 추론
2. 멀티-에이전트 오케스트레이션
3. 오프라인 + 온라인 하이브리드

## BRANCH B: 보수적 관점 (현실주의)

### 1. 강력한 경쟁자 현황

- Claude Code: SWE-bench 72.5%, 8개월만에 시장 선도, 300K+ 비즈니스 고객
- Cursor: Fortune 500의 50% 사용, $500M ARR, 가장 빠른 SaaS 성장
- GitHub Copilot: Microsoft 백업, Office 365 번들

### 2. 로컬 실행의 근본적 한계

- 모델 성능: 로컬 7B/13B vs Claude 72.5% (큰 격차)
- 하드웨어 장벽: 16GB+ VRAM 필요
- 설치/유지 복잡성: Docker, CUDA, 수동 업데이트
- 기업 판매 어려움: SLA, 지원, 공급업체 신뢰도

### 3. 주요 리스크 3가지

| 리스크 | 심각도 |
|-------|--------|
| 성능 경쟁 불가능 (Claude 72.5% vs 로컬 50%) | 🔴 Critical |
| 멀티-에이전트 오케스트레이션 복잡성 | 🔴 Critical |
| 기업 판매 어려움 | 🔴 Critical |

## BRANCH A vs B 비교

**현재 시장 데이터 지지**: B (보수적) 65% > A (낙관적) 35%

### Optimistic이 타당한 신호
- EU AI Act 2026년 시행
- Cline 1.5M 사용자 (수요 검증)
- 멀티-에이전트 2026년 주류화

### Cautious가 타당한 신호
- Claude Code 압도적 성장 속도
- Cursor Fortune 500 50% 침투
- 로컬 모델 성능 격차 (단기 해소 불가)
- 개발자 심리: 편의성 > 비용 절감

## PRD 방향을 위한 핵심 인사이트 3가지

1. **"진정한 로컬 실행"은 5-10% 틈새 시장** → 프리미엄 niche 포지셔닝
2. **하이브리드(로컬+클라우드)가 실용적 방향** → 비용 30-40% 절감 + 프라이버시
3. **멀티-에이전트 오케스트레이션 = 유일한 게임 체인저**

### 피해야 할 전략
- ❌ "Claude Code를 더 싸게 하는 버전"
- ❌ "오픈소스를 더 쉽게 하는 래퍼"
- ❌ "비용 절감이 주요 가치 제안"
